# 🎮 Sorteador de Campeões - League of Legends

Aplicativo mobile desenvolvido em React Native (Expo) que sorteia um campeão aleatório de League of Legends, exibindo sua imagem, título, tipos e nível de dificuldade.

Os dados são consumidos da API oficial **Data Dragon** da Riot Games.

## ✨ Funcionalidades

- Sorteia um campeão aleatório a cada toque no botão
- Exibe imagem, nome e título do campeão
- Mostra os tipos (Lutador, Mago, Tanque, etc.)
- Indica o nível de dificuldade (de 1 a 10)

## 🛠️ Tecnologias

- React Native
- Expo
- Data Dragon API (Riot Games)
